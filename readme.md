# SIGE - TP3 (Seminario)

Este proyecto corresponde al Trabajo Práctico 3 de la materia Seminario de Práctica (Licenciatura en Informática).  
Consiste en un prototipo funcional en Java que simula el funcionamiento de un sistema de gestión escolar.

## Funcionalidades implementadas
- Registro de usuarios, materias, asistencias y calificaciones
- Menú funcional en consola
- Manejo de excepciones
- Uso de POO (encapsulamiento, abstracción, etc.)

## Requisitos
- JDK 17
- NetBeans IDE

## Compilación
Abrir el proyecto en NetBeans, ejecutar la clase `SIGE.java`.
